## What The Extension Actually Does
1. **Reads text you type** - Only in chat input fields on Claude, ChatGPT, and T3Chat
2. **Appends text** - Adds "NO EMDASHES" to your messages before you send them
3. **Stores one setting** - Your on/off preference, stored locally in Chrome's storage (never leaves your device)

That's it. That's the entire functionality.

## Permissions Explained

### `storage`
- **Why we need it:** To remember if you've toggled the extension on or off
- **What we store:** A single boolean value (true/false)
- **Where it's stored:** In Chrome's local storage on YOUR device only

### `activeTab`
- **Why we need it:** To read and modify the text in chat input fields
- **What we access:** Only the input field where you type messages
- **When we access it:** Only when you're actively using Claude, ChatGPT, or T3Chat
